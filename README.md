# File Sharing App

A simple full-stack file sharing application similar to Google Drive.

## Features
- User authentication (Register/Login)
- Upload files (PDF, Images, CSV)
- Bulk upload support
- View uploaded files with metadata
- Share files with specific users
- Secure file download with authorization
- JWT-based authentication

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT
- File Upload: Multer
